

# Node Version Manager（Node 版本管理器）



nvm (Linux、Unix、OS X)
  + https://github.com/creationix/nvm
  + 产用命令：
    - nvm install node （安装最新版本的node）
    - nvm use node （使用指定版本的node）


nvm-windows (Windows)
  + https://github.com/coreybutler/nvm-windows
  + 常用命令：
    - nvm version
    - nvm install latest
    - nvm install 版本号
    - nvm uninstall 版本号
    - nvm list
    - nvm use 版本号
